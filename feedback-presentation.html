CORE FEATURES

核心功能 - 後端 API

🔌8 個 REST API 端點

  • GET /api/feedbacks - 取得所有回饋
  • GET /api/feedbacks/{id} - 取得單一回饋
  • GET /api/feedbacks/station/{id} - 依車站查詢
  • POST /api/feedbacks - 新增回饋
  • DELETE /api/feedbacks/{id} - 刪除回饋
  • GET /api/feedbacks/statistics - 所有統計
  • GET /api/feedbacks/statistics/{id} - 車站統計
  • GET /api/feedbacks/average-rating/{id} - 平均評分
4
DATA MODEL

資料模型設計

📊Feedback 實體

  • id (Long) - 主鍵
  • rating (Integer) - 評分 1-5 星
  • comment (String) - 評論內容
  • passengerName (String) - 乘客姓名
  • station (Station) - 關聯車站
  • createdAt (LocalDateTime) - 建立時間

🔗關聯設計

Many-to-One
關聯類型

多個回饋 → 一個車站

使用 @ManyToOne 註解
外鍵: station_id
