## ⚡ 重要：必須在 Workshop 前 3 天完成

### 1️⃣ 註冊 Bob 試用帳號

請前往 Bob 官網註冊免費試用帳號：

🔗 **https://bob.ibm.com**

**註冊步驟**：
1. 點擊「取得免費試用」
2. **建議使用 Google 帳號註冊**（或使用公司 email）
3. 完成註冊後，請檢查信箱並點擊驗證連結
4. 確認可以登入 Bob 網站

💡 **試用版提供**：40 Bobcoins，30 天試用期，足夠完成所有 Workshop 練習

---

### 2️⃣ 安裝必要軟體

#### **IBM Bob IDE**（必要）

下載連結：🔗 **https://bob.ibm.com/download**

- **macOS**：下載 `.pkg` 檔案並安裝
- **Windows**：下載 `.exe` 檔案並安裝
- **Linux**：下載對應的 `.deb` 或 `.rpm` 檔案

安裝完成後，請啟動 Bob 並使用您註冊的帳號登入。

#### **Git**（必要）

下載連結：🔗 **https://git-scm.com/downloads**

安裝後請在終端機執行以下指令驗證：
```bash
git --version
```

---

### 3️⃣ 測試環境

請完成以下測試，確保環境正常：

**測試 Bob**：
1. 啟動 Bob 應用程式
2. 使用註冊帳號登入
3. 開啟聊天面板（點擊側邊欄 Bob 圖示）
4. 輸入：`列出當前目錄的檔案`
5. 點擊「執行」，確認可以正常運作

**測試 Git**：
```bash
git --version
```
應該顯示 Git 版本號（例如：git version 2.x.x）
