## ✅ 必做事項（Workshop 前 3 天完成）

### 1. 註冊 Bob 帳號
- 前往：https://bob.ibm.com
- 點擊「取得免費試用」
- **建議使用 Google 帳號註冊**（或公司 email）
- 確認收到驗證信並完成驗證

### 2. 安裝軟體

#### Bob IDE（必要）
- 下載：https://bob.ibm.com/download
- 選擇對應作業系統版本安裝
- 啟動後使用註冊帳號登入

#### Git（必要）
- 下載：https://git-scm.com/downloads
- 安裝後驗證：`git --version`

## 🧪 快速測試

### 測試 Bob
1. 啟動 Bob 應用程式
2. 使用帳號登入
3. 開啟聊天面板
4. 輸入：`列出當前目錄的檔案`
5. 點擊「執行」確認可正常運作

### 測試 Git
```bash
git --version
```
